Project Choropleth Map

Indonesia Covid-19 Spread Choropleth Map for Province

In this project i want to visualize covid-19 spread in Indonesia using Choropleth Map. It is a map in which different regions are shaded according to the variable that represents an aggregate summary for each area. Two important things that require for choropleth maps are geojson data of Indonesia with the geopraphic boundaries and variables for shaded which is data of covid increasing cases each province in Indonesia.

I got shp file that has geographic boundaries information from data.humadata.org. the shp file need to reduce for simplicity of the data because it need more time to process bigger data. Data with covid-19 information each province in Indonesia downloaded from kawalcorona API. It have variable for coloring each province. both data must have a variable that have exactly same code to specify where to match the values of locations.

After all data has been prepared, last step is to plot the choropleth map
