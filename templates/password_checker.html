Project Password Checker

An App to Check if Your Password is Secure

There's a website named haveibeenpwned.com that can check if your password has ever been hacked but it's not pretty safe to check your password that way because when you send your password to a server there are still a possibility that someone can intercept your password. I have built password checker that would help to check password in a more secure way.

The way this password checker work is it will send request to 'api.pwnedpasswords.com' + the password we want to check. It wont send the password as a plaintext because this is not how the api.pwnedpasswords works instead we want to do something called hashing. A hashing algorithm is a mathematical function that make your password to a fixed size. hashing is a one way function, means we can't use our output to find the input. so if someone get your hashed password no function to reverse this hashed password back to your plaintext password. there are many hash function but this API using hash function call SHA1. SHA1 will take the password and turn into hexadecimal number with 40 digits long. Only the first five characters of your hash password will send to the API.

When we send request, API will response with list of five characters + the rest of 35 characters hashed passwords that have exactly same five characters with password you send. This program will check if there is the rest 35 characters from the list that exactly same with yours to conclude password has ever been hacked or not.
